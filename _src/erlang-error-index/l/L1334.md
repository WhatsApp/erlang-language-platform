# L1334 - Native Record Undefined

## Error

```erlang
-module(main).
-export_record([point]).
%%             ^^^^^ error: native record point undefined
```

## Explanation

The error occurs when an [`-export_record`](https://www.erlang.org/eeps/eep-0079)
attribute names a record that has not been defined in the module.

Only native records that are defined in the current module can be exported. To
fix the error, define the record with `-record(#point {...}).` before exporting
it, or remove it from the export list.
